# ZKChAT-HD
ZeKChAt est une application de messagerie moderne conçue avec une identité africaine forte. Elle offre une expérience fluide et premium : conversations en temps réel, statuts, appels, personnalisation des thèmes et assistance IA intégrée. Développée en HTML/CSS/JS pur avec Supabase comme backend, ZeKChAt prouve qu'on peut créer des produits tech.
